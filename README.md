**VHub CMS (Developer Workspace Dashboard)**

**Position:** Fullstack Developer (Personal)

**Công nghệ:** ReactJS, Vite, Ant Design, Framer Motion, Node.js, Express, Firebase (Firestore), Supabase.

**Mô tả:**

- Thiết kế và phát triển hệ thống Content Management System (CMS) chuyên biệt để quản lý dự án cá nhân, đồng bộ bài đăng Blog và theo dõi nhân sự.
- Hệ thống tập trung tối ưu hóa quy trình quản trị nội dung cho Developer với giao diện hiện đại, Dashboard thống kê đa chiều và trải nghiệm người dùng mượt mà.
- Dự án là minh chứng cho khả năng xây dựng, xử lý luồng dữ liệu bảo mật (JWT/OAuth) và đóng gói sản phẩm Fullstack hoàn chỉnh (Production-ready).

**Chức năng chính:**

- **Authentication & Security:** Tích hợp Supabase Auth (đăng nhập Google, GitHub, Email) với hệ thống phân quyền an toàn, hỗ trợ chế độ Demo (Viewer Mode - chỉ xem, chặn toàn bộ thao tác CRUD).
- **Project Intelligence Hub:** Dashboard quản lý danh mục dự án, thống kê Repo Analytics (Commits, Views, Stars) thông qua các biểu đồ trực quan.
- **TikTok Content Sync:** Chức năng tự động fetch và đồng bộ hàng loạt video từ TikTok để chuyển đổi thành các định dạng bài viết Blog chuyên nghiệp.
- **UI/UX & Animations:** Tích hợp thư viện Framer Motion để tạo ra các hiệu ứng chuyển cảnh (transitions) và tương tác (micro-interactions) tinh tế, chuyên nghiệp.
- **Responsive:** Áp dụng tư duy Mobile-first, đảm bảo hệ thống hiển thị và thao tác hoàn hảo trên mọi loại thiết bị (Mobile, Tablet, Desktop).

## 🛠️ Hướng dẫn cài đặt local

1. **Clone Repo**:
   ```bash
   git clone https://github.com/Vinhdev04/VHub-CMS.git
   ```
2. **Cài đặt Dependency**:
   ```bash
   cd backend && npm install
   cd ../frontend && npm install
   ```
3. **Cấu hình Môi trường**: Tạo file `.env` theo `example.env` và khởi động.
